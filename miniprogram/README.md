# QuestMind 小程序

人工智能训练师（三级）刷题小程序，UI 基于 QuestMind 设计稿。

## 功能

- **题库**：判断题、单选题、多选题、操作题（合并原简答题 + 代码题）
- **错题本**：记录错题、统计分布
- **我的**：学习数据与个人设置

## 题目数据

题目由 `scripts/build_miniprogram_data.py` 自动生成：

- 理论题：`qustions/理论题/*.md`
- 操作题：`qustions/理论题/操作题.md`（题目与评分表合并解析）
- 代码填空：`QB/*/xxx.ipynb`（每空附带分值）
- 简答模板：`素材/*.docx`（自动读取 docx 已有格式）

```bash
python scripts/build_miniprogram_data.py
```

将 docx 答题素材放入 `素材/` 目录后重新运行构建，即可在操作题界面显示预填模板。

## 开发

使用微信开发者工具打开 `miniprogram` 目录即可预览。

> 参考答案字段暂为空，待后续补充后可自动启用判分。

## 题目评论（讨论区）

每道理论题、操作题底部均有「讨论区」，支持发表评论、回复他人，并展示全部评论。

### 云开发配置（多人互通）

1. 微信开发者工具 → **云开发** → 开通环境
2. 云数据库新建集合 `comments`（权限可设为「仅管理端可写」，由云函数写入）
3. 右键 `cloudfunctions/commentApi` → **上传并部署：云端安装依赖**
4. 重新编译小程序

未开通云开发时，评论会保存在本机本地，仅当前设备可见；开通后自动切换为云端互通。
