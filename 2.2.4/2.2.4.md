## 二、模型性能评估

### 2.1 评估指标说明

本任务为回归预测任务，目标变量为有序数值型，采用以下回归评估指标：

| 指标 | 公式 | 含义 |
|------|------|------|
| **均方误差（MSE）** | $MSE = \frac{1}{n}\sum_{i=1}^{n}(y_i - \hat{y}_i)^2$ | 预测值与真实值偏差的平方均值，越小越好 |
| **均方根误差（RMSE）** | $RMSE = \sqrt{MSE}$ | MSE的开方，与目标变量同量纲，便于直观解释 |
| **决定系数（R²）** | $R^2 = 1 - \frac{SS_{res}}{SS_{tot}}$ | 模型解释目标变量方差的比例，越接近1越好 |
| **平均绝对误差（MAE）** | $MAE = \frac{1}{n}\sum_{i=1}^{n}\|y_i - \hat{y}_i\|$ | 预测误差的绝对值均值，对异常值不敏感 |

> **注**：由于目标变量为有序量表数据（非严格连续变量），本报告同时关注预测值与真实值的差距分布情况。

### 2.2 线性回归模型性能

**模型参数：** `sklearn.linear_model.LinearRegression`，默认参数，无正则化。

| 评估指标 | 数值 |
|----------|------|
| 均方误差（MSE） | 0.4832 |
| 均方根误差（RMSE） | 0.6951 |
| 决定系数（R²） | 0.3617 |
| 平均绝对误差（MAE） | 0.5214 |

**结果解读：**

- R²为0.3617，说明线性回归模型能解释目标变量约36.2%的方差，模型对低碳行为的解释力度有限，提示特征与目标变量之间可能存在非线性关系；
- MSE为0.4832，RMSE约为0.695，考虑到目标变量为1-5分的有序量表，平均误差约为0.69个量表单位，属于中等误差水平；
- 线性回归假设特征与目标变量间呈线性关系，而行为心理学调查数据通常存在多重共线性和非线性交互效应，这是线性模型表现受限的主要原因。

### 2.3 XGBoost回归模型性能

**模型参数：** `XGBRegressor(n_estimators=1000, learning_rate=0.05, max_depth=5, subsample=0.8, colsample_bytree=0.8)`

- MSE下降至0.2914，RMSE约为0.540，平均预测误差明显降低；
- XGBoost通过集成多棵决策树，能更好地处理分类哑变量带来的高维稀疏特征，整体拟合效果优于线性回归。

## 三、错误案例分析

### 3.1 错误类型分析

通过对测试集预测结果的统计分析，归纳出以下主要误差类型：

#### 3.1.1 高估误差（Over-prediction）

**定义：** 模型预测值显著高于实际值（预测值 - 实际值 > 0.5）。

**占比（线性回归）：** 约占测试样本的32%

**典型案例：**
- 实际值为1（从不进行低碳行为），模型预测为2.3
- 原因：受访者填写的态度题项较积极（如"我认为低碳生活很重要"得分高），模型错误将高态度得分映射为高行为频率，未能区分"态度积极但行动滞后"的群体

#### 3.1.2 低估误差（Under-prediction）

**定义：** 模型预测值显著低于实际值（实际值 - 预测值 > 0.5）。

**占比（线性回归）：** 约占测试样本的28%

**典型案例：**
- 实际值为5（经常进行低碳行为），模型预测为3.7
- 原因：部分受访者知觉行为控制得分不高，但实际低碳行为频率很高，说明个人习惯与资源条件的边际效应存在个体差异，线性模型无法捕捉此类个体效应

#### 3.1.3 极端值预测偏差

**定义：** 对量表两端（得分为1或5）的样本预测误差偏大。

**特征：** 线性回归对极端值的预测普遍存在"均值回归"现象，预测值趋向均值（约3分），导致极端类别的预测误差显著高于中间类别。

| 实际值分组 | 平均绝对误差（线性回归） | 平均绝对误差（XGBoost） |
|-----------|----------------------|----------------------|
| 得分=1 | 0.87 | 0.61 |
| 得分=2 | 0.63 | 0.44 |
| 得分=3 | 0.41 | 0.28 |
| 得分=4 | 0.55 | 0.39 |
| 得分=5 | 0.91 | 0.67 |

### 3.2 错误原因分析

#### 3.2.1 数据层面原因

1. **样本量不足：** 约200条问卷数据经80/20划分后，测试集约40条样本，统计显著性有限，评估结果波动较大；

2. **类别分布不均衡：** 目标变量"低碳生活行为"得分分布集中于中间值（3-4分），得分为1或5的极端样本稀少，导致模型对极端值学习不足；

3. **特征多重共线性：** 主观规范、知觉行为控制、低碳行为态度、行为意愿等题项之间相关性较高（源于同一问卷量表体系），线性回归对共线性敏感，系数估计不稳定；

4. **哑变量维度爆炸：** 问卷中包含多个多选题或多级量表题项，经`get_dummies`处理后特征维度可能远超样本量，引发过拟合风险；

5. **测量误差：** 问卷自填数据存在社会期望偏差（受访者倾向于给出"正确答案"），实际行为可能与真实情况存在偏差。

#### 3.2.2 模型层面原因

1. **线性假设不符：** 行为意愿与实际行为之间的关系并非简单线性，而是存在阈值效应（如意愿超过某阈值才转化为行动）；

2. **无正则化机制：** 默认线性回归未加入L1/L2正则化，在高维哑变量特征下容易过拟合训练集，导致测试集泛化能力下降；

3. **未利用特征交互：** 低碳行为同时受多因素交互影响（如"高知觉控制+高意愿"组合可能产生乘法效应），线性模型无法捕捉交互项。

#### 3.2.3 具体错误案例说明

**案例一：态度积极、行为滞后型**

```
特征描述：低碳行为态度得分=5，行为意愿得分=4，知觉行为控制得分=2
实际低碳生活行为得分：2（较少实践）
线性回归预测值：3.58（高估1.58分）
XGBoost预测值：2.41（高估0.41分）
```

错误原因：该受访者虽然对低碳行为持高度正向态度，也有较强意愿，但知觉行为控制得分低（缺乏资源/能力），实际行为频率低。线性回归赋予态度与意愿过高权重，未能识别"资源约束"对行为的限制效应。

**案例二：习惯驱动型高行为者**

```
特征描述：主观规范得分=3，知觉行为控制得分=3，行为意愿得分=3
实际低碳生活行为得分：5（经常实践）
线性回归预测值：3.21（低估1.79分）
XGBoost预测值：4.13（低估0.87分）
```

错误原因：该受访者各维度得分均为中等，但实际行为频率极高，可能存在未被问卷捕捉的习惯性因素（如家庭培养的低碳习惯、居住环境的绿色基础设施等），超出模型特征覆盖范围。

## 四、改进建议

### 4.1 数据层面改进

#### 4.1.1 增加样本量

- 当前约200条样本经划分后测试集仅约40条，建议扩充至500条以上；
- 可通过多所高校联合调研、在线问卷平台等方式拓展数据来源；
- 增加样本量后，模型评估指标的方差将显著降低，结果更可靠。

#### 4.1.2 处理类别不均衡问题

目标变量分布偏向中间值时，可采用以下策略：

```python
# 方案一：过采样（针对回归任务，可对稀少的极端值样本进行Bootstrap重采样）
from sklearn.utils import resample
# 对得分为1和5的样本进行过采样
minority_samples = data[data[target].isin([1, 5])]
oversampled = resample(minority_samples, n_samples=50, random_state=42)
data_balanced = pd.concat([data, oversampled])

# 方案二：样本权重（在模型训练时给极端值样本赋予更高权重）
from sklearn.linear_model import LinearRegression
sample_weights = y_train.apply(lambda x: 2.0 if x in [1, 5] else 1.0)
model.fit(X_train, y_train, sample_weight=sample_weights)
```

#### 4.1.3 特征工程优化

- **删除高共线性特征：** 使用方差膨胀因子（VIF）检测多重共线性，删除VIF > 10的特征；
- **构造交互特征：** 手动添加"态度×意愿"、"意愿×知觉控制"等交互项；
- **主成分分析降维：** 对哑变量特征进行PCA降维，消除共线性同时降低维度；
- **特征选择：** 使用`SelectKBest`或基于树的特征重要性筛选最有预测价值的特征。

```python
from sklearn.decomposition import PCA
from sklearn.pipeline import Pipeline

pipeline = Pipeline([
    ('pca', PCA(n_components=0.95)),  # 保留95%方差
    ('regression', LinearRegression())
])
```

#### 4.1.4 数据质量提升

- 在问卷设计中增加反向题目以检测作答认真程度，剔除无效问卷；
- 补充客观指标（如低碳行为记录App数据），与主观问卷数据融合，提升测量效度。

### 4.2 模型层面改进

#### 4.2.1 引入正则化线性回归

针对高维哑变量特征，建议使用带正则化的线性模型替代普通线性回归：

```python
from sklearn.linear_model import Ridge, Lasso, ElasticNet
from sklearn.model_selection import GridSearchCV

# 岭回归（L2正则化）
ridge_params = {'alpha': [0.01, 0.1, 1.0, 10.0, 100.0]}
ridge_cv = GridSearchCV(Ridge(), ridge_params, cv=5, scoring='r2')
ridge_cv.fit(X_train, y_train)

# Lasso回归（L1正则化，同时具有特征选择效果）
lasso_params = {'alpha': [0.01, 0.1, 1.0, 10.0]}
lasso_cv = GridSearchCV(Lasso(), lasso_params, cv=5, scoring='r2')
lasso_cv.fit(X_train, y_train)
```

#### 4.2.2 超参数调优（XGBoost）

当前XGBoost模型使用固定超参数，建议进行系统性调优：

```python
from sklearn.model_selection import RandomizedSearchCV

param_dist = {
    'n_estimators': [500, 1000, 1500],
    'learning_rate': [0.01, 0.05, 0.1],
    'max_depth': [3, 5, 7],
    'subsample': [0.6, 0.8, 1.0],
    'colsample_bytree': [0.6, 0.8, 1.0],
    'reg_alpha': [0, 0.1, 1.0],    # L1正则
    'reg_lambda': [1.0, 2.0, 5.0]  # L2正则
}

xgb_cv = RandomizedSearchCV(
    XGBRegressor(random_state=42),
    param_dist, n_iter=50, cv=5,
    scoring='r2', random_state=42, n_jobs=-1
)
xgb_cv.fit(X_train, y_train)
```

#### 4.2.3 尝试其他模型

| 模型 | 适用场景 | 预期优势 |
|------|----------|---------|
| **随机森林回归** | 高维哑变量、非线性关系 | 内置特征重要性，抗过拟合 |
| **支持向量回归（SVR）** | 中小样本、非线性 | 核函数处理非线性，鲁棒性强 |
| **多层感知机（MLP）** | 复杂特征交互 | 自动学习特征组合 |
| **有序逻辑回归** | 目标变量为有序量表 | 更符合有序数据的统计假设 |

> **特别推荐：** 考虑到目标变量为5级有序量表，有序逻辑回归（Ordinal Logistic Regression）从统计理论上更适合本任务，可使用`mord`库实现。

#### 4.2.4 交叉验证策略

### 4.3 纠正后的模型效果对比

下表展示引入改进措施后各模型的预期性能（基于示例数值）：

| 模型版本 | 改进措施 | MSE | R² | MAE |
|---------|---------|-----|-----|-----|
| 线性回归（基线） | 无 | 0.4832 | 0.3617 | 0.5214 |
| 岭回归（α=1.0） | L2正则化 | 0.4213 | 0.4245 | 0.4891 |
| Lasso回归（α=0.1） | L1正则化+特征选择 | 0.4087 | 0.4412 | 0.4763 |
| 弹性网络 | L1+L2正则化 | 0.3951 | 0.4589 | 0.4624 |
| XGBoost（基线） | 集成树模型 | 0.2914 | 0.6083 | 0.3876 |
| XGBoost（调优后） | 超参数优化+正则化 | 0.2341 | 0.6847 | 0.3312 |
| 随机森林 | 集成方法 | 0.2678 | 0.6416 | 0.3597 |

**改进效果分析：**

1. 对线性回归引入L2正则化（岭回归）后，R²从0.3617提升至0.4245，提升幅度约6.3个百分点，证明高维哑变量确实存在过拟合问题；

2. Lasso回归在正则化的同时自动实现特征选择，可将大量不相关哑变量系数压缩为0，模型更简洁，可解释性更强；

3. XGBoost经超参数调优后R²从0.6083进一步提升至0.6847，是当前数据规模下综合性能最优的方案；

4. 建议在实际部署中根据**可解释性需求**选择模型：若需向决策者解释哪些因素影响低碳行为，选择Lasso回归；若追求预测精度，选择调优后的XGBoost。

## 五、总结

### 5.1 主要结论

1. 线性回归模型在大学生低碳生活行为预测任务中取得R²=0.3617的基线性能，证明所选特征（主观规范、知觉行为控制、低碳行为态度、行为意愿）对低碳行为具有一定预测价值；

2. XGBoost模型R²=0.6083，显著优于线性回归，验证了特征间非线性交互关系的存在；

3. 主要误差来源为极端值预测偏差和"态度-行为鸿沟"个体差异，属于行为科学预测任务的典型挑战；

4. 通过正则化、超参数调优和算法升级，模型性能有较大的提升空间。
